¿Cuál es el lenguaje estándar para aplicar estilos a las páginas web?

  • Java
  • PHP
  • CSS

¿Cuál es la mejor manera de aplicar estilos en una página web?

  • Incluir los estilos en las etiquetas HTML para que se carguen y ejecuten antes.
  • Incluirlos en un fichero externo vinculado a ese fichero HTML.
  • Incluirlos en la cabecera para agruparlos en un mismo sitio en la página.

¿Qué significa que un elemento está 'deprecated' en HTML4?

  • Que las versiones más modernas de los navegadores no lo entenderán bien y no se podrán visualizar correctamente.
  • Que ya no forma parte de las especificaciones de la versión 4.
  • Que ha cambiado de nombre ese elemento en la versión 4.
